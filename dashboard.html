Company Management System For Manpower Security Company

  • Insert Data
  • Edit Data
  • Delete Data
  • Calculate Client Payments
  • Calculate Supplier Payments
  • Calculate Guard Salaries
  • Record Guard Attendance
  • Generate Reports

The aim of this project is to develop a platform independent software application that can assist the company in storing and manage the data that can be used for decision making purposes. The system will provide assistance to the manager in inserting information, editing information, deleting information, c alculating the payment to be payment by clients, calculating the payment to be payment to suppliers, calculating salary of guards, recording attendance of guards, as well as generating reports.
